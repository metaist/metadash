# [metadash](http://metaist.com/dash) - a lightweight cookie-based dashboard
metadash is a single web-page that displays a list of links that are persisted in a cookie.

It's an [example program](http://metaist.com/blog/2012/02/code-review-metadash.html) for
introducing HTML, JavaScript, and CSS (via [jQuery](http://jquery.com/) and [LESS](http://lesscss.org/)).
